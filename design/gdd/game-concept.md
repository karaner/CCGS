# 游戏概念：《牌神》

*创建日期: 2026-03-31*
*状态: 草稿*

---

## 概述

《牌神》是一款以战场环境控制为核心的回合制牌组构建roguelike（DBG）。玩家通过打出卡牌在战场上编织火焰、暗影、血迹、陷阱等持续性领域，配合卡牌升级路径构建独特Build。核心乐趣在于"战场的每一格都是你策划的死亡迷宫"——当领域叠加、Combo触发，玩家感受到精心策划后碾压敌人的智谋快感。

不同于传统DBG只是"打出更强的牌"，《牌神》的战场本身就是玩家最强的武器。

## 详细规则

### 1. 战斗回合结构

**每回合流程**：
1. **回合开始** — 领域效果触发（灼烧扣血、中毒叠加等）
2. **抽牌阶段** — 抽5张牌（手牌上限10，超出弃牌）
3. **行动阶段** — 消耗能量打出卡牌（每回合3能量，能量不跨回合）
4. **敌人行动阶段** - 敌人按意图行动（攻击/防御/技能）
5. **回合结束** — 弃牌至手牌上限，触发领域结算

**能量系统**：
- 每回合固定3能量
- 卡牌消耗1-3能量不等
- 能量不跨回合保留

### 2. 环境领域系统

**领域类型**（MVP阶段）：
| 领域 | 效果 | 叠加规则 |
| ---- | ---- | ---- |
| 血迹 | 无直接效果，被踩时触发"流血"状态（持续掉血3回合） | 可叠加4层，每层+1伤害/回合 |
| 火焰 | 每回合造成2点伤害，持续3回合 | 可叠加，叠加后灼烧伤害+1 |
| 暗影 | 敌人移动消耗额外1格；领域内敌人受到伤害+15% | 不同叠加层数有不同加成 |
| 陷阱 | 敌人首次进入触发：眩晕1回合或造成3点伤害 | 触发后消失 |

**领域交互表**：
| 组合 | 结果 | 说明 |
| ---- | ---- | ---- |
| 火焰 + 血迹 | 蒸发 | 血迹消失，火焰伤害+2 |
| 火焰 + 暗影 | 暗焰 | 暗影伤害加成+5% |
| 暗影 + 血迹 | 腐化 | 流血伤害+1 |

**领域持续时间**：
- 血迹：永久（被踩后消失）
- 火焰：3回合
- 暗影：4回合
- 陷阱：触发后消失

### 3. 卡牌系统

**卡牌结构**：
- 名称、能耗（0-3）、效果描述、领域操作

**卡牌类型**：
- **攻击牌** — 造成伤害，可能附带领域放置
- **防御牌** — 获得护甲，护甲持续当前战斗
- **领域牌** — 核心机制，在战场上放置/增强领域
- **功能牌** — 抽牌、能量回复、移除负面状态等

**升级系统**：
每张卡有2-3个升级路径，升级后卡牌名称/效果/能耗可能改变：
- **路径A：伤害强化** — 伤害+50%，可能改变能耗
- **路径B：领域强化** — 附带更强领域效果
- **路径C：效率优化** — 能耗-1，效果不变

**升级时机**：
- 战斗胜利后选择奖励时
- 休整点可花费金币移除卡牌或升级

### 4. 职业系统（仅战士完成MVP）

**战士**：
- **领域媒介**：血迹、火焰
- **核心机制**：
  - 鲜血充能（血迹越多，攻击越强）
  - 护甲共鸣（护甲可转化为反击伤害）
- **初始卡组**：10张基础攻击/防御牌
- **节奏感**：前期偏防守积累血迹，后期爆发输出

### 5. 敌人意图系统

敌人每回合显示下一回合的行动意图：
| 意图 | 显示图标 | 效果 |
| ---- | ---- | ---- |
| 攻击 | 剑 | 对玩家造成X点伤害 |
| 防御 | 盾 | 获得X点护甲 |
| 技能 | 特殊图标 | 释放特殊技能（范围攻击、Buff等） |

### 6. 肉鸽地图结构

**地图节点类型**：
| 类型 | 出现概率 | 说明 |
| ---- | ---- | ---- |
| 普通战斗 | 40% | 2-4个基础敌人 |
| 精英战斗 | 15% | 1个精英敌人，奖励更好 |
| 奖励节点 | 20% | 卡牌奖励/金币/回复 |
| 事件节点 | 15% | 随机事件（二选一后果） |
| 商店 | 10% | 消耗金币购买卡牌/回复 |
| Boss | 1次 | 战斗3后出现 |

**路径选择**：
- 每次选择1-2个前方节点
- 休整点后难度提升
- 部分路径需要特定条件解锁

## 公式

### 1. 伤害计算

**基础伤害公式**：
```
最终伤害 = 基础伤害 × (1 + 伤害加成%) + 固定伤害
```

**领域增伤**：
```
最终伤害 = 基础伤害 × (1 + 领域增伤%) + 领域固定伤害
```

**流血伤害（血迹触发）**：
```
每回合流血伤害 = 血迹层数 × (1 + 玩家流血强化%)
持续回合 = 3回合（固定）
```

### 2. 护甲系统

**护甲计算**：
```
本回合护甲 = 上回合护甲 + 本回合新增护甲
护甲不会自动衰减（除非被击破或特殊机制）
```

**护甲转化反击（战士职业特性）**：
```
反击伤害 = 本回合获得护甲 × 0.5
（需要护甲共鸣天赋激活）
```

### 3. 领域叠加

**血迹叠加**：
```
叠加层数上限 = 4层
层数N时的流血伤害 = N × 1点/回合
```

**火焰叠加**：
```
叠加层数上限 = 无上限
层数N时的灼烧伤害 = N × 2点/回合
首次叠加：伤害+2惩罚（蒸发血迹效果）
```

**暗影叠加**：
```
层数1：敌人移动消耗+1格，伤害+15%
层数2：额外+5%伤害（累计+20%）
层数3：额外+5%伤害（累计+25%）
层数4：敌人所有属性-10%
```

### 4. Combo触发

**Combo系数**：
```
Combo伤害 = 单卡伤害 × (1 + 0.25 × (触发领域数 - 1))
最多叠加3次，即最高+75%伤害
```

**领域共鸣触发条件**：
- 同一回合打出2张以上领域相关卡牌
- 敌人站在多个领域重叠处

### 5. 卡牌升级数值

| 升级类型 | 伤害变化 | 能耗变化 |
| ---- | ---- | ---- |
| 伤害强化路径 | +50% | 可能+1 |
| 领域强化路径 | ±0% | ±0% |
| 效率优化路径 | ±0% | -1 |

### 6. 敌人强度缩放

**普通敌人（战斗N）**：
```
生命值 = 基础生命值 × (1 + 0.1 × (N-1))
攻击力 = 基础攻击力 × (1 + 0.08 × (N-1))
```

**精英敌人**：
```
生命值 = 普通敌人 × 2.5
攻击力 = 普通敌人 × 1.2
```

**Boss**：
```
生命值 = 普通敌人 × 8
攻击力 = 普通敌人 × 1.5
特殊技能：每战斗2回合释放一次
```

### 7. 资源 economy

**金币获取**：
```
普通战斗胜利 = 10 + 5 × 战斗序号
精英战斗胜利 = 30 + 10 × 战斗序号
Boss战斗胜利 = 100（金币+卡牌奖励）
```

**休整费用**：
```
升级单张卡牌 = 50金币
移除单张卡牌 = 75金币
回复至满血 = 30金币
```

## 边缘情况

### 1. 手牌溢出
- **触发条件**：抽牌后手牌超过10张
- **处理方式**：强制弃牌至10张，弃牌随机选择
- **UI提示**：显示"手牌已满，已弃置X张牌"

### 2. 能量溢出
- **触发条件**：打出卡牌后剩余能量无法使用
- **处理方式**：能量不跨回合保留，回合结束时清零
- **无补偿机制**：鼓励玩家规划能量使用

### 3. 领域重叠冲突
- **火焰 + 血迹**：触发"蒸发"效果，血迹消失，火焰伤害+2
- **多个同类型领域叠加**：按叠加规则处理（如火焰灼烧伤害叠加）
- **领域放置在已有领域上**：替换或叠加（根据领域类型决定）

### 4. 敌人移动到已有领域
- **血迹**：敌人踩到血迹，触发流血debuff
- **火焰**：敌人踩到火焰，触发灼烧debuff
- **暗影**：敌人进入暗影区域即享受增伤debuff（持续整场战斗）
- **陷阱**：触发眩晕或伤害效果

### 5. 玩家无法行动
- **触发条件**：手牌为空且无能量打牌
- **处理方式**：自动跳过行动阶段，直接进入敌人行动阶段
- **稀有情况**：仅在手牌质量极差时发生

### 6. 敌人死于领域伤害
- **触发时机**：回合开始阶段（领域效果触发时）
- **处理方式**：正常获得奖励，战斗结束
- **经验获取**：领域击杀与直接击杀同等计算

### 7. Boss战中途死亡
- **触发条件**：Boss被击败
- **处理方式**：战斗立即结束，显示Boss掉落

### 8. 领域持续时间到期
- **血迹**：永久存在，除非被踩或被蒸发
- **火焰**：持续3回合后自动消失，回合开始时减少1回合
- **暗影**：持续4回合后自动消失
- **陷阱**：触发后立即消失

### 9. 卡牌移除
- **触发条件**：玩家主动移除或特殊效果强制移除
- **处理方式**：卡牌从牌组和手牌中移除（手牌中移除进入弃牌堆）
- **限制**：每局游戏移除卡牌次数有限（休整点最多3次）

### 10. 负面状态叠加
- **流血**：可叠加4层，每层独立计时
- **眩晕**：不可叠加，覆盖刷新持续时间
- **护甲减益**：不可叠加，覆盖刷新

## 依赖

### 系统依赖关系

| 系统 | 依赖 | 说明 |
| ---- | ---- | ---- |
| **战斗系统** | 卡牌系统、领域系统、敌人AI | 战斗是核心玩法，依赖卡牌效果和领域结算 |
| **卡牌系统** | 战斗系统、升级系统 | 卡牌通过战斗发挥作用 |
| **领域系统** | 战斗系统、敌人AI | 领域影响战斗流程和敌人行为 |
| **升级系统** | 卡牌系统 | 升级改变卡牌数值和效果 |
| **敌人AI** | 战斗系统、领域系统 | 敌人需要理解战场状态（领域、意图） |
| **肉鸽地图** | 战斗系统、奖励系统、休整系统 | 地图节点触发战斗和奖励 |

### 双向依赖说明

| 系统A ←→ 系统B | 依赖描述 |
| ---- | ---- |
| 战斗 ↔ 卡牌 | 战斗消耗卡牌，卡牌效果改变战斗状态 |
| 战斗 ↔ 领域 | 战斗触发领域效果，领域效果影响战斗伤害 |
| 战斗 ↔ 敌人AI | 战斗是玩家与敌人的互动，敌人AI决定战斗节奏 |
| 升级 ↔ 卡牌 | 升级改变卡牌属性，卡牌是升级的载体 |
| 肉鸽地图 ↔ 战斗 | 地图提供战斗遭遇，战斗结果影响地图进度 |
| 肉鸽地图 ↔ 休整 | 地图包含休整点，休整影响后续战斗能力 |

### 被依赖项（需要先实现）

| 系统 | 被依赖次数 | 说明 |
| ---- | ---- | ---- |
| **战斗系统** | 5 | 所有系统都需要战斗接口 |
| **卡牌系统** | 3 | 升级和UI都依赖卡牌数据 |
| **数据驱动架构** | 2 | 卡牌效果和敌人属性需要数据驱动 |

### 实现优先级建议

1. **核心循环**：战斗系统 → 卡牌系统 → 领域系统
2. **支持系统**：敌人AI → 肉鸽地图 → 升级系统
3. **外围系统**：UI系统 → 音频系统 → 存档系统

## 一句话介绍

> 你不是出牌攻击——你是战场的编织者。在回合制DBG（牌组构建roguelike）中，用卡牌重塑地形、召唤元素、留下陷阱。你的Build不是抽到什么牌决定，而是你把每张牌升级成什么方向。当Build成型、环境叠加，你精心设计的战场成为毁灭敌人的舞台。

## 核心幻想

**"你是战场的主宰者。"**

你用卡牌编织战场——留下燃烧的痕迹、蔓延的暗影、精心布置的陷阱。敌人步步踏入你设计的死亡迷宫，而你静静等待Build成型的瞬间。当元素叠加、领域重叠、Combo触发——你看着敌人被淹没在你一手策划的毁灭中，感受到真正的智谋与力量的爆发。

这是"运筹帷幄"的快感，也是"精心策划后碾压对手"的满足。

---

## 独特卖点

**就像杀戮尖塔的战斗，但战场本身才是你最强的武器。**

不同于传统DBG只是"打出更强的牌"：
- **环境即武器** — 你铺设的火焰、暗影、陷阱不是附属效果，而是Build的核心载体
- **升级即Build** — 同一张牌，升级方向不同，Build完全不同。没有无所谓的好牌，只有方向不同的Build
- **领域叠加** — 敌人的脚下不只是受伤的格子——是你精心设计的死亡迷宫

### 关键动态（涌现的玩家行为）

- 玩家尝试卡牌组合来发现联动
- 玩家主动操控战场位置来最大化领域效果
- 玩家与社区分享发现的"强力Combo"
- 玩家反复游玩来探索不同的升级路径和职业Build

### 核心机制（我们构建的系统）

1. **回合制战斗系统** — 能量+抽牌+行动经济，每回合3能量，抽5张牌
2. **环境战场系统** — 持续领域（火、暗影、陷阱等）跨回合存在，可以叠加和交互
3. **卡牌升级系统** — 每张卡牌有多个升级路径；选择决定Build方向
4. **职业系统** — 战士/法师/盗贼，每个有独特的领域媒介和代价类型
5. **肉鸽跑图结构** — 随机地图、遭遇选择、奖励选择、休整点

## 核心循环

### 30秒循环（回合制战斗的核心操作）

**玩家最常做的事：**
1. 手牌5张
2. 决定打出哪几张（通常1-3张）
3. 点击卡牌打出 → 伤害/敌人效果/领域放置
4. 敌人意图结算
5. 回合结束 → 领域效果触发

**什么让它感觉好：**
- 卡牌打出有重量感（屏幕震动、打击感、数字弹出）
- 领域放置有策略感（选择位置很重要）
- Combo触发很爆炸（多效果连锁）

### 5-15分钟循环（一场战斗）

**一场战斗遭遇：**
- Boss/精英战是谜题——找到处理他们机制的Build
- 普通遭遇让你测试和完善正在成型的Build
- "再来一回合"心理：你离触发Combo只差1回合

### 30-120分钟循环（一次完整跑图）

**一次完整跑图：**
```
[地图节点选择] → [遭遇（战斗/奖励/事件）] → [休整/升级] → [下一个节点] → ... → [BOSS] → [局次总结]
```

**自然停止点：**
- 任何遭遇后（可以退出，继续地图）
- 休整点（存档/退出）
- Boss击败后

**什么让他们回来：**
- "如果我尝试不同的卡牌升级路径会怎样？"
- "我就差一点点就能用那个Build赢了"
- 新内容（未来：新职业、新遭遇）

### 长期进度

- **解锁系统**：赢得局次解锁更高难度、新起始卡牌
- **收集**：卡牌变体和视觉升级
- **精通**：完全了解所有卡牌交互
- **没有"终点"**：肉鸽=无限重玩价值

## 游戏支柱

### 支柱1：环境即武器

卡牌不只是造成伤害——它们重塑战场。你的Build由你如何操控环境定义。

*设计测试：在"造成10点伤害"和"造成6伤害 + 在敌人脚下留下火领域（持续灼烧伤害 + 将来可引爆）"之间，这个支柱说选后者。*

### 支柱2：每张牌都可以是一个Build

卡牌升级是Build的核心驱动力。同一张牌，不同升级方向 = 完全不同的玩法。

*设计测试：如果两个升级路径产生类似结果但体验不同，选体验更独特的那条路。*

### 支柱3：每个职业有完全不同的手感

战士、法师、盗贼不只是数值和图片不同——他们的环境操控方式、代价类型、节奏感都根本不同。

*设计测试：如果一个设计让两个职业玩起来感觉差不多，毙掉。*

### 支柱4：成长要有爆发感

Build成型的瞬间要有强烈的正反馈。Combo触发、伤害爆发、逆转翻盘——这些时刻是玩家记住游戏的瞬间。

*设计测试：如果一个机制让玩家"慢慢变强但没有明显爆发点"，重新设计。*

### 支柱5：策略深度大于运气

环境控制赋予玩家主动权。选择比运气重要。

*设计测试：如果一个游戏感觉像"我抽到正确的牌了吗？"，它失败了。如果它感觉像"我如何战略性地利用我抽到的牌？"，它成功了。*

### 反支柱（这个游戏不是什么）

- **不是集卡游戏** — 你不需要大量卡牌；深度大于广度
- **不是运气游戏** — 环境控制赋予玩家主动权；选择 > 运气
- **不是背板游戏** — 每局都不同；决策能力 > 记忆最优解
- **不只是打伤害** — 战斗不只是换血；环境操控是核心
- **不是竞技多人** — 纯PvE体验

## 风险和开放问题

### 设计风险
- 环境系统可能对核心循环清晰度来说太复杂 — **尽早原型验证**
- 卡牌升级多样性可能导致"最优路径"主导策略 — 需要测试
- Build强度节奏可能感觉糟糕（太慢或太快） — 测试中调整

### 技术风险
- 领域叠加/重叠交互可能复杂 — 在原型中验证
- 卡牌效果系统需要灵活的数据驱动设计 — 使用组合而非继承

### 市场风险
- DBG类型已有 established 竞争对手（杀戮尖塔、怪物火车） — 用环境操控钩子差异化
- 个人开发者1年 timeline — 必须控制范围

### 范围风险
- 3职业 × 多升级路径 × 平衡 = 大量QA面
- 环境领域的视觉效果需要大量美术/动画工作

### 开放问题
- **环境领域如何交互？**（火领域 + 冰领域 = 蒸汽？蒸汽 = 致盲？）
- **玩家移动如何运作？**（只靠职业技能，还是自由定位？）
- **什么触发"Build完成"时刻？**（需要清晰的强度信号）

## MVP定义

**核心假设**：玩家觉得"环境控制 + 卡牌升级 = Build"循环在30分钟以上的局次中很有吸引力。

**MVP所需**：
1. **一个完整职业**（战士）— 所有卡牌、升级路径、职业机制
2. **环境领域系统** — 至少3种领域类型（血迹/火焰/暗影）
3. **10+个遭遇** — 普通战斗、精英战斗、1个Boss
4. **基础肉鸽地图** — 3-4节点路径、2个休整点、Boss
5. **卡牌升级系统** — 每张卡至少2个升级路径
6. **核心战斗循环** — 能量、抽牌、打出、敌人意图、回合结束

**MVP明确不包括**：
- 第2和第3职业（法师、盗贼）
- 高级Combo和连锁反应
- 完整解锁进度系统
- 多个难度级别

### 范围层级

| 层级 | 内容 | 功能 | 时间线 |
| ---- | ---- | ---- | ---- |
| **MVP** | 1职业、30张卡、10个遭遇、1 Boss | 核心战斗 + 领域系统 + 升级 | 3个月 |
| **垂直切片** | 3职业、60张卡、完整地图、3 Boss | 所有职业 + 更多遭遇 | 6个月 |
| **Alpha** | 3职业、80张卡、完整内容 | 所有功能，需要打磨/平衡 | 9个月 |
| **完整愿景** | + 发售后新职业、完整打磨 | 所有内容 + 打磨 + 1个新职业 | 12个月 |
