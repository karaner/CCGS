## 一句话介绍

> 你不是出牌攻击——你是战场的编织者。在回合制DBG（牌组构建roguelike）中，用卡牌重塑地形、召唤元素、留下陷阱。你的Build不是抽到什么牌决定，而是你把每张牌升级成什么方向。当Build成型、环境叠加，你精心设计的战场成为毁灭敌人的舞台。

## 核心幻想

**"你是战场的主宰者。"**

你用卡牌编织战场——留下燃烧的痕迹、蔓延的暗影、精心布置的陷阱。敌人步步踏入你设计的死亡迷宫，而你静静等待Build成型的瞬间。当元素叠加、领域重叠、Combo触发——你看着敌人被淹没在你一手策划的毁灭中，感受到真正的智谋与力量的爆发。

这是"运筹帷幄"的快感，也是"精心策划后碾压对手"的满足。

---

## 独特卖点

**就像杀戮尖塔的战斗，但战场本身才是你最强的武器。**

不同于传统DBG只是"打出更强的牌"：
- **环境即武器** — 你铺设的火焰、暗影、陷阱不是附属效果，而是Build的核心载体
- **升级即Build** — 同一张牌，升级方向不同，Build完全不同。没有无所谓的好牌，只有方向不同的Build
- **领域叠加** — 敌人的脚下不只是受伤的格子——是你精心设计的死亡迷宫

## 玩家体验分析（MDA框架）

### 目标美学（玩家感受到的）

| 美学 | 优先级 | 如何实现 |
| ---- | ---- | ---- |
| **感官刺激** | 3 | 卡牌打出特效、领域效果视觉、爆发时刻 |
| **幻想** | 2 | 三个职业的独特幻想：战士的血怒、法师的元素掌控、盗贼的暗影舞步 |
| **叙事** | 2 | 基于局次的进度，遭遇之间的叙事片段 |
| **挑战** | 1 | Boss战、精英敌人、Build优化谜题 |
| **社交** | N/A | 无多人内容 |
| **探索** | 2 | 隐藏联动、意外卡牌组合、秘密遭遇房间 |
| **表达** | 1 | 卡牌升级路径定义玩家身份；Build多样性 |
| **休闲** | N/A | 策略深度意味着参与度，不是放松 |

### 关键动态（涌现的玩家行为）

- 玩家尝试卡牌组合来发现联动
- 玩家主动操控战场位置来最大化领域效果
- 玩家与社区分享发现的"强力Combo"
- 玩家反复游玩来探索不同的升级路径和职业Build

### 核心机制（我们构建的系统）

1. **回合制战斗系统** — 能量+抽牌+行动经济，每回合3能量，抽5张牌
2. **环境战场系统** — 持续领域（火、暗影、陷阱等）跨回合存在，可以叠加和交互
3. **卡牌升级系统** — 每张卡牌有多个升级路径；选择决定Build方向
4. **职业系统** — 战士/法师/盗贼，每个有独特的领域媒介和代价类型
5. **肉鸽跑图结构** — 随机地图、遭遇选择、奖励选择、休整点

| ---- | ---- | ---- |

| **能力** | 掌握职业机制、发现联动、优化伤害 | 核心 |

### 玩家类型吸引度（ Bartle 分类）

- [x] **成就者** — 如何吸引：Build优化、局次完成、解锁进度
- [x] **探索者** — 如何吸引：隐藏联动、元素交互深度、未知的卡牌组合
- [ ] **社交者** — 如何吸引：N/A — 纯单人
- [ ] **杀手/竞争者** — 如何吸引：N/A — PvE为主

### 心流设计

- **上手曲线**：第一局教卡牌打出 → 第二局引入升级选择 → 第三局揭示职业机制
- **难度缩放**：精英敌人和Boss需要针对性Build；普通敌人允许实验
- **反馈清晰度**：Combo触发、伤害数字、视觉效果清晰传达Build成功
- **失败恢复**：肉鸽=随时重开；死亡感觉是"下一局会更好"而非"浪费时间"

## 核心循环

### 30秒循环（回合制战斗的核心操作）

**玩家最常做的事：**
1. 手牌5张
2. 决定打出哪几张（通常1-3张）
3. 点击卡牌打出 → 伤害/敌人效果/领域放置
4. 敌人意图结算
5. 回合结束 → 领域效果触发

**什么让它感觉好：**
- 卡牌打出有重量感（屏幕震动、打击感、数字弹出）
- 领域放置有策略感（选择位置很重要）
- Combo触发很爆炸（多效果连锁）

### 5-15分钟循环（一场战斗）

**一场战斗遭遇：**
- Boss/精英战是谜题——找到处理他们机制的Build
- 普通遭遇让你测试和完善正在成型的Build
- "再来一回合"心理：你离触发Combo只差1回合

### 30-120分钟循环（一次完整跑图）

**一次完整跑图：**
```
[地图节点选择] → [遭遇（战斗/奖励/事件）] → [休整/升级] → [下一个节点] → ... → [BOSS] → [局次总结]
```

**自然停止点：**
- 任何遭遇后（可以退出，继续地图）
- 休整点（存档/退出）
- Boss击败后

**什么让他们回来：**
- "如果我尝试不同的卡牌升级路径会怎样？"
- "我就差一点点就能用那个Build赢了"
- 新内容（未来：新职业、新遭遇）

### 长期进度

- **解锁系统**：赢得局次解锁更高难度、新起始卡牌
- **收集**：卡牌变体和视觉升级
- **精通**：完全了解所有卡牌交互
- **没有"终点"**：肉鸽=无限重玩价值

## 游戏支柱

### 支柱1：环境即武器

卡牌不只是造成伤害——它们重塑战场。你的Build由你如何操控环境定义。

*设计测试：在"造成10点伤害"和"造成6伤害 + 在敌人脚下留下火领域（持续灼烧伤害 + 将来可引爆）"之间，这个支柱说选后者。*

### 支柱2：每张牌都可以是一个Build

卡牌升级是Build的核心驱动力。同一张牌，不同升级方向 = 完全不同的玩法。

*设计测试：如果两个升级路径产生类似结果但体验不同，选体验更独特的那条路。*

### 支柱3：每个职业有完全不同的手感

战士、法师、盗贼不只是数值和图片不同——他们的环境操控方式、代价类型、节奏感都根本不同。

*设计测试：如果一个设计让两个职业玩起来感觉差不多，毙掉。*

### 支柱4：成长要有爆发感

Build成型的瞬间要有强烈的正反馈。Combo触发、伤害爆发、逆转翻盘——这些时刻是玩家记住游戏的瞬间。

*设计测试：如果一个机制让玩家"慢慢变强但没有明显爆发点"，重新设计。*

### 支柱5：策略深度大于运气

环境控制赋予玩家主动权。选择比运气重要。

*设计测试：如果一个游戏感觉像"我抽到正确的牌了吗？"，它失败了。如果它感觉像"我如何战略性地利用我抽到的牌？"，它成功了。*

### 反支柱（这个游戏不是什么）

- **不是集卡游戏** — 你不需要大量卡牌；深度大于广度
- **不是运气游戏** — 环境控制赋予玩家主动权；选择 > 运气
- **不是背板游戏** — 每局都不同；决策能力 > 记忆最优解
- **不只是打伤害** — 战斗不只是换血；环境操控是核心
- **不是竞技多人** — 纯PvE体验

## 参考和灵感

| 参考 | 我们借鉴什么 | 我们有何不同 | 为什么重要 |
| ---- | ---- | ---- | ---- |
| 杀戮尖塔 | DBG战斗循环、卡牌升级系统、肉鸽地图 | 环境/战场操控作为核心机制，不只是卡牌效果 | 验证DBG+肉鸽框架；我们的钩子是"战场即武器" |
| 炉石传说 | 职业身份、卡牌交互深度、法力系统 | 无攻击回合（纯PvE）、领域代替随从 | 我们追求的职业幻想和策略深度 |
| 第七史诗 | Build成长满足感、角色成长体验 | 卡牌型（而非角色型）Build、肉鸽代替抽卡 | "Build成型"的感觉是我们的核心情感目标 |
| 怪物火车 | 多区域战斗、领域封锁策略 | 回合制（而非即时）、升级路径多样性 | 验证环境控制作为可行核心机制的案例 |

**非游戏灵感**：桌游领域控制（Risk、Terraforming Mars）、万智牌的堆叠交互

## 风险和开放问题

### 设计风险
- 环境系统可能对核心循环清晰度来说太复杂 — **尽早原型验证**
- 卡牌升级多样性可能导致"最优路径"主导策略 — 需要测试
- Build强度节奏可能感觉糟糕（太慢或太快） — 测试中调整

### 技术风险
- 领域叠加/重叠交互可能复杂 — 在原型中验证
- 卡牌效果系统需要灵活的数据驱动设计 — 使用组合而非继承

### 市场风险
- DBG类型已有 established 竞争对手（杀戮尖塔、怪物火车） — 用环境操控钩子差异化
- 个人开发者1年 timeline — 必须控制范围

### 范围风险
- 3职业 × 多升级路径 × 平衡 = 大量QA面
- 环境领域的视觉效果需要大量美术/动画工作

### 开放问题
- **环境领域如何交互？**（火领域 + 冰领域 = 蒸汽？蒸汽 = 致盲？）
- **玩家移动如何运作？**（只靠职业技能，还是自由定位？）
- **什么触发"Build完成"时刻？**（需要清晰的强度信号）

## MVP定义

**核心假设**：玩家觉得"环境控制 + 卡牌升级 = Build"循环在30分钟以上的局次中很有吸引力。

**MVP所需**：
1. **一个完整职业**（战士）— 所有卡牌、升级路径、职业机制
2. **环境领域系统** — 至少3种领域类型（血迹/火焰/暗影）
3. **10+个遭遇** — 普通战斗、精英战斗、1个Boss
4. **基础肉鸽地图** — 3-4节点路径、2个休整点、Boss
5. **卡牌升级系统** — 每张卡至少2个升级路径
6. **核心战斗循环** — 能量、抽牌、打出、敌人意图、回合结束

**MVP明确不包括**：
- 第2和第3职业（法师、盗贼）
- 高级Combo和连锁反应
- 完整解锁进度系统
- 多个难度级别

### 范围层级

| 层级 | 内容 | 功能 | 时间线 |
| ---- | ---- | ---- | ---- |
| **MVP** | 1职业、30张卡、10个遭遇、1 Boss | 核心战斗 + 领域系统 + 升级 | 3个月 |
| **垂直切片** | 3职业、60张卡、完整地图、3 Boss | 所有职业 + 更多遭遇 | 6个月 |
| **Alpha** | 3职业、80张卡、完整内容 | 所有功能，需要打磨/平衡 | 9个月 |
| **完整愿景** | + 发售后新职业、完整打磨 | 所有内容 + 打磨 + 1个新职业 | 12个月 |
